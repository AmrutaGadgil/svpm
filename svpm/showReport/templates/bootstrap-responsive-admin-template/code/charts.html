Modify Distribution Range



Click for description
This control gives some runtime flexibility to decide threshold values in the pie chart. By default, business risk is perceived as 'medium' if its value lies between 4-7. Any value above it is perceived as 'high', and any value below it is perceived as 'low'.

To set these two thresholds, change the two pointers above. The value will also get reflected in the legends below the pie chart.
